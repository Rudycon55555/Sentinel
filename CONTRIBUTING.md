# Contributing to Sentinel

Thank you for your interest in contributing to Sentinel!  
This project is built to empower developers to create secure, modular, full‑stack system applications in Rust — and your contributions help make that vision stronger.

## Ways to Contribute

- Fix bugs or improve existing modules  
- Add new Frontend, Backend, or Middlend features  
- Improve documentation or examples  
- Build extensions using the Sentinel Extensions API  
- Help with testing, validation, or security hardening  

## Development Setup

1. Install Rust (latest stable)
2. Install Tauri prerequisites (Node.js, system dependencies)
3. Clone the repository:
   ```bash
   git clone https://github.com/Rudycon55555/Sentinel.git
   ```
4. Build the project:
   ```bash
   cargo build
   ```
## Code Style
  - Use Rust 2021 edition
  - Keep modules small, focused, and well‑documented
  - Prefer composition over inheritance
  - Avoid unnecessary unsafe code
  - Follow idiomatic Rust patterns
## Pull Requests
Before submitting a PR:

- Ensure the project builds successfully
- Run `cargo fmt` and `cargo clippy`
- Add documentation for new features
- Include tests when possible
## Reporting Issues
If you find a bug or have a feature request, please open an issue on GitHub with:

  - A clear description
  - Steps to reproduce (if applicable)
  - Expected vs actual behavior
